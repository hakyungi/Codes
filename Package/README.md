# Package

OpenMV H7 / H7 Plus 카메라에 I/F 보드를 장착하여 TOF(적외선 거리 측정센서) 센서 3개와 9 DOF IMU(BNO055) 센서를 연결 한 샘플 코드입니다.

I2C 주소만 틀리다면 선을 병렬로 연결하여 센서 추가가 가능합니다.(예 MLX90614 비접촉식 온도센서)

I/F 보드와 EV3 또는 SPike Prime과 연결한 예입니다.
